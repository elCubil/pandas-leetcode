# Pandas LeetCode Solutions

This repository contains my solutions to LeetCode problems using **Python** and **Pandas**.

The main objective of this repository is to strengthen the data manipulation skills required for **Data Engineering** and **Data Analytics** roles.

## Topics Covered

- Data filtering
- GroupBy operations
- Aggregations
- Ranking
- Window-like operations
- Merge / Join
- Shift
- Conditional transformations
- Datetime manipulation
- Data cleaning

## Repository Structure

Each file contains:

- The original input data as a Pandas DataFrame.
- A complete solution using Pandas.
- The final expected output.

This allows every exercise to be executed independently.

## Technologies

- Python
- Pandas
- NumPy

## Goal

I am building this repository as part of my preparation for **Data Engineer** positions.

The focus is on writing clear, efficient, and vectorized Pandas solutions while improving problem-solving skills through LeetCode exercises.

More Data Engineering projects (ETL pipelines, SQL, Docker, orchestration, and cloud tools) will be published in separate repositories.
